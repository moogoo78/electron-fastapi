## Getting Started

### Prerequisites

* npm
* python3

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/moogoo78/electron-flask
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. set env for Windows
   ```sh
   npm config set script-shell "powershell"
   ```

## Usage

build fastapi as python exe file
```sh
npm run package-py-win
```
